Chromeが重いときに試したい5つの対策

Chromeを使っていると、いつの間にかタブが増えて、PC全体が重く感じることがあります。だからといって、作業中のタブを片っ端から閉じる必要はありません。まずはChromeの標準機能を試し、それでも足りないときにタブ管理ツールを使うのがおすすめです。

1. まずはメモリセーバーを試す

Chromeの「設定」→「パフォーマンス」で、メモリセーバーをオンにします。しばらく使っていないタブは自動で休止し、もう一度開いたときにページが読み込まれます。

タブを休止するタイミングは設定から選べます。仕事で使うメールやチャットなど、常に動かしておきたいサイトは対象外にできます。

なお、メモリセーバーと省エネモードは別の機能です。メモリセーバーは使っていないタブを休止し、省エネモードはバッテリーの消費を抑えます。

2. 自分で選んだタブだけ休止する

休止するタブを自分で確かめたい場合は、タブ管理用の拡張機能が便利です。

Tab Reliefでは、使用中・固定済み・音声再生中のタブは休止しません。休止したタブも閉じず、そのままの場所に残します。必要になったら、いつもどおりタブを開くだけです。

3. あとで使うタブは保存して閉じる

数日後にまた使う資料や、いったん中断する作業のタブは、開いたままにせずワークスペースとして保存できます。

保存するときに「旅行の予約」「研究資料」のような名前を付けておけば、必要になったときにまとめて開き直せます。作業の区切りでタブを閉じられるので、普段から開いておくタブの数も減らせます。

4. もう見ないタブをまとめて閉じる

YouTubeや通販サイトを見ていると、同じようなタブが何枚も残ることがあります。タイトルやURLに含まれる文字で候補を絞り、いらないタブだけをまとめて閉じれば、すっきり片付きます。

タブの休止と違い、閉じたタブは一覧から消えます。実行する前に、閉じるタブを一度確認しておくと安心です。

5. 使っていない拡張機能を見直す

Chromeのタスクマネージャーを開くと、どのタブや拡張機能がメモリを使っているか確認できます。使っていない拡張機能が見つかったら、chrome://extensionsから無効にするか削除します。

拡張機能は、インストールされているだけで常に同じ量のメモリを使うわけではありません。名前だけで判断せず、実際の数字を見てから整理するのが確実です。

本当に軽くなったか確認する

変更する前と後で、ChromeのタスクマネージャーやWindowsのメモリ表示を比べます。動画やWebアプリを開いているタブは、文字中心のページより多くのメモリを使うことがあります。

減る量は開いているページによって変わるため、「1タブで必ず何MB減る」とは言えません。できるだけ同じタブを開いた状態で比べると、効果が分かりやすくなります。
