Chromeのメモリ使用量を減らす5つの方法

読了目安5分 · Chrome 121以降対応

Chromeは多くの人が使うブラウザの中でも特にメモリを消費します。開いているタブ、拡張機能、バックグラウンドのプロセスがそれぞれRAMを使い、8GB以下のマシンでは体感できるほど重くなります。ただし、作業中のタブを閉じなくても軽くする方法はあります。

1. Chrome標準の「メモリセーバー」を使う

Chrome 108以降は「メモリセーバー」機能を搭載しています。設定 → パフォーマンスでオンにすると、しばらく使っていないタブが自動で休止されメモリが解放されます。タブを開き直すと通常通り再読み込みされます。

制限: どのタブを休止するかを細かく制御できず、ウィンドウごと丸ごと保存してから片付ける機能はありません。

2. 拡張機能でタブを手動スリープする

ピン留めしたタブや音声再生中のタブ、まだ読んでいる調べ物は休止させたくない、という場合は専用の拡張機能で制御できます。Tab Reliefはメモリセーバーと同じchrome.tabs.discard()APIを使用しているため、タブは瞬時に復帰します。違いは「何を休止するか」を自分で決められる点です。

3. ウィンドウをワークスペースとして保存してから閉じる

メモリを大量消費しがちなのは、プロジェクト用に開いた20〜40枚のタブをまとめたウィンドウです。そのウィンドウを「ワークスペース」として名前をつけて保存すれば、タブを全部閉じてスッキリさせた後、必要なときにまとめて復元できます。これだけで300〜600MBが解放されることがあります。

4. 重複タブや溜まったタブをまとめて片付ける

同じURLを誤って2回開いた重複タブは気づきにくいけれど積み重なります。一致タブの整理機能を使えば、たとえば「YouTubeのタブをまとめて閉じる」「Amazonのタブだけ選んで片付ける」といった操作を確認画面付きで安全に行えます。

5. 拡張機能の数を見直す

有効になっているChrome拡張機能は、それぞれバックグラウンドで動作しメモリを消費します。chrome://extensionsを開き、1か月以上使っていない拡張機能はオフにしましょう。無効化するだけでメモリ使用量はゼロになります。

実際にどれくらい変わるか

目安として: 15〜20枚の非アクティブタブをスリープすると200〜500MB程度回収できます。使い終わったプロジェクトウィンドウ(30タブ前後)を閉じると600MB以上回収できることもあります。数値はどのサイトを開いているかによって大きく変わります。メール・SNS・動画など、JavaScriptを多く使うページは特にメモリを消費します。
