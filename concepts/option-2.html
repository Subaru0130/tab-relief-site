What it actually does

Four tools for a browser that gets messy every day.

The page should make a heavy Chrome user understand what Tab Relief does, what stays protected, and how to start safely.

01

Sleep inactive tabs

Use Chrome's native tab discard behavior to rest background tabs, reduce load, and bring a tab back when the user opens it again.

02

Auto Sleep rules

Let Tab Relief quietly sleep safe inactive tabs after a chosen interval, starting from a sensible 15-minute default.

03

Workspace restore

Save a whole messy window as a named workspace, clean up, and restore the saved set later when the task comes back.

04

Close matching clutter

Pick common presets or type words to find matching tabs by title, domain, or URL, then close them in a controlled review step.
